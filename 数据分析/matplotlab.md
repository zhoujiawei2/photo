# matplotlib

axis轴，指的是x轴或者是y轴这种轴

```python
	from matplotlib import pyplot as plt
	x=[1,2,5,6]
	y=[3,4,1,7]
	plt.plot(x,y)
	plt.show()
```  
figure图形图标的意思，在这里指画的图
```python
	fig=plt.figure(figsize=(20,8),dpi=80)
```  
